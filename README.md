# Projeto Instagram React

Este é um projeto inspirado no Instagram, desenvolvido durante o curso da Driven Education. Ele foi criado com **React** e utiliza componentes, estados e props para renderizar posts, stories e sugestões de seguidores dinamicamente.

## Funcionalidades

- Layout semelhante ao Instagram Web (desktop)
- Stories horizontais com animação de hover
- Feed com posts, likes, comentários e possibilidade de salvar posts
- Sidebar fixa com usuário principal e sugestões de seguidores
- Atualização dinâmica do nome e foto de perfil do usuário
- Likes interativos (curtir e descurtir)
- Componentização com React (Navbar, Stories, Feed, Post, Sidebar, User, Suggestions)

## Tecnologias utilizadas

- React
- HTML / CSS
- JavaScript
- npm (Node.js)
